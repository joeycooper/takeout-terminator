# 纠结终结者 (Takeout Terminator) 🤖🍱

由章鱼哥 (Joey) 发起，大将军小派 (Pi) 荣誉开发。

这是一款专门为现代人设计的点餐焦虑拯救神器。

## 核心功能
- **三大心情频道**：全都要、要身材、要快乐。
- **神话级老虎机**：顺滑的转动动画，精准的餐点锁定。
- **一键App唤起**：支持美团外卖、饿了么、大众点评 App 一键搜索。
- **环境感应**：根据当前时间（早中晚宵夜）和天气（通过 API 自动获取）给出人性化建议。
- **健康监督**：记录点餐历史，防止重复摄入高热量食物。
- **自定义菜单**：支持在线编辑和后端持久化保存。

## 部署说明
本系统由前端 HTML + 后端 Node.js 服务组成。
1. 安装 Node.js
2. 运行 `npm start` 启动后端服务
3. 使用 Web 服务器（如 Nginx 或 serve）托管前端页面
